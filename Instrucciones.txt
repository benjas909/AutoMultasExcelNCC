# AutoMultasExcel

Desarrollado en Python 3.13, Windows 10 Enterprise 22H2

## Prerrequisitos

- Python 3.xx

- Openpyxl

## Antes de ejecutar por primera vez

- Instalar la versión más reciente de Python desde la Microsoft Store.

- Abrir una ventana de Powershell en la carpeta donde se encuentre este programa:
  - En la carpeta, en un espacio vacío, hacer click derecho y elegir, "Abrir en Terminal"
  - En caso de que no aparezca esta opción, hacer Shift + Click derecho y seleccionar "Abrir la ventana de PowerShell aquí"

- Ejecutar el comando "python ./get-pip.py" en powershell, esperar a que finalice

- Al finalizar, ejecutar "pip install openpyxl"

- En caso de no obtener errores, se puede ejecutar el programa

## Ejecución del programa

- En una pestaña de PowerShell ubicada en la carpeta que contiene el programa, ejecutar el comando "python ./test.py"

- Se abrirá una ventana con una casilla que se debe seleccionar en caso de ser necesaria la importación de la hoja de tickets.
  Al seleccionar esta casilla, se activará el botón para elegir el archivo a importar.

- Además, se debe seleccionar el archivo de multas y la ubicación y nombre del archivo de salida del programa.

- Cuando aparezca el mensaje de "Archivo Guardado", el archivo estará listo para abrir en la ubicación elegida.
